# email-guesser
Email Guesser Test

## Frontend

- Nextjs app
- Fetch url hardcoded to http://localhost:5000/convert in pages/index.js

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
